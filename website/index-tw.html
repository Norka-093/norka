嘿 我是 NORKA

一位前端開發人員專注於使用 HTML、CSS、JavaScript 和 React 等框架，來打造響應式、使用者友善且視覺上引人入勝的 Web 體驗。透過簡潔的程式碼和直覺的設計將想法變成現實。

關於我
作為一名志在成為前端開發人員的新人，雖然尚未擁有正式的工作經驗，但透過勤奮自學，我已建立穩固的前端開發基礎。我通過 HTML、CSS、JavaScript 等核心技術，能夠獨立設計簡潔且具美感的靜態網頁，並打造流暢、直覺且具吸引力的使用者體驗與互動介面。

此外，我對電腦硬體組裝充滿熱情。從精挑細選相容的處理器、顯示卡等設備，到精心規劃內部佈局與線路整理，我注重細節並享受動手實踐的過程，結合專業知識與精確執行。

憑藉對技術的熱忱與專業精神，我有信心能迅速適應前端開發角色，在實務中持續學習與成長，並為團隊帶來創新思維與積極能量。
